## A difference

Change the function so that it *returns* the difference between number `a` and the sum of `b` and `c`.

Hint

Use the `return` operator, do not try to output the values to the console.

**Sample Input 1:**

```
33 2 1
```

**Sample Output 1:**

```
30
```

**Sample Input 2:**

```
30 20 10
```

**Sample Output 2:**

```
0
```
